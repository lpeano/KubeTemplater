# Welcome to KubeTemplater Documentation

KubeTemplater is a lightweight, config-driven Kubernetes operator that dynamically renders and applies Kubernetes resources using Go templates defined within your `ConfigMap`s.

This documentation provides a comprehensive guide to understanding, installing, and using the KubeTemplater operator.

## 📖 Table of Contents

- **[Getting Started](./getting-started.md)**: Learn how to install and set up the operator in your cluster.
- **[How It Works](./how-it-works.md)**: Understand the core concepts and reconciliation logic of the operator.
- **[Advanced Features](./features.md)**: Discover advanced features like the force-replace strategy for immutable resources.
- **[Examples](./examples.md)**: Find practical examples for common use cases.
